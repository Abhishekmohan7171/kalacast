# Image Assets

Please save the uploaded images with the following names in this directory:

## Required Images:

1. **actors-illustration.png**
   - The illustration showing three actors with cameras and equipment
   - Used in: Mission Banner component
   - Location: `src/assets/images/actors-illustration.png`

2. **producer-tools-ui.png**
   - The producer tools interface showing filters and actor profiles
   - Used in: Producer Tools component
   - Location: `src/assets/images/producer-tools-ui.png`

3. **director-silhouette.png**
   - The dark silhouette of a director with a hat
   - Used in: About component
   - Location: `src/assets/images/director-silhouette.png`

## How to Save:

1. Save each uploaded image with the exact filename listed above
2. Place them in this directory: `d:\Angular\kalacast\src\assets\images\`
3. The app will automatically load them once saved

## Image Format:

- Recommended format: PNG (for transparency support)
- Alternative: JPG/JPEG (if no transparency needed)
- The code will work with either format
